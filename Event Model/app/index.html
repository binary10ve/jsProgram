Lesson #1

What is Javascript Event Model?

When the user does something on a webpage an event takes place. Not always the events are triggered by User, there are also some events like the load event that fires when a page has been loaded.

Javascript allows you to attach listeners to such events. These listeners are executed when those event occurs. Event could be of multiple Types.

Event Types:

  • Mouse Events

  • Keyboard Events

  • Touch Events

  • Form Events

  • DOM Events

  • Custom Events

Events handlers can be attached in multiple ways.

In HTML:

<button id="my-button" onclick="myFunction()">Click Me!!</button>

In Javascript:

domElement.onclick=function(){...};

In Javascript using the addEventListener() method:

domElement.addEventListener("click", function(){...});

Example:

   
                      

                        function myFunction(){
                          alert('I am clicked')
                        }
                      

                      <button onclick="myFunction"id="my-button">
                      Click Me!!.
                      </button>
